Mapping PML 1 Class Properties to PML 2

  • All PML 1 classes and properties are listed but not PML 2. Please refer to PML 2 Ontology Specification for all PML 2 classes and properties.
  • If no corresponding PML 2 property listed for a PML 1 property, it is deprecated.
  • Some PML 1 class properties are mapped to "complex" property in PML 2. For example, property "URL" of class "Format" in PML 1 is now mapped to "hasURL" property of PML 2 class "Information" which is the range of property "hasDescription" of PML 2 class "Format". So for PML 2, the "URL" of "Format" is shown as "hasDescription->hasURL".
